# 异构计算 作业七

> OpenCL二维卷积计算
>
> 1. 编写代码，采用 CPU 实现二维卷积计算；
> 2. 采用循环展开、SIMD 指令优化 CPU 执行性能；
> 3. 编写 OpenCL 代码，采用 GPU 实现二维卷积计算；
> 4. 采用存储器优化、工作项并行优化异构执行性能；
> 5. 基于稀疏矩阵乘法优化方法，进一步提升卷积计算性能；
> 6. 改变数据类型，对比Int8/16/32与 Float16/32数据计算的性能差异。

设计了以下四种程序：

1. 未手动优化的 CPU 程序；

   > 该程序作为参考实现。
2. 使用 AVX2 指令集手动向量化的程序；

   > 我的电脑不支持 AVX-512，故只测试了 AVX2。
3. 未手动优化的 OpenCL 程序；
4. 使用 $16 \times 16$ 的 Tile，且卷积核使用稀疏表示的 OpenCL 程序。

   $$
