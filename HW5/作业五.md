# 异构计算 作业五

> 采用图像存储器，编写二维图像旋转的 OpenCL 程序。基本要求：
>
> 1. 参考资料 \[1\] 中提供了样例代码，编写 OpenCL Kernel；
> 2. 编写相同功能的 CPU 程序；
> 3. 选取至少两个不同尺寸的图像，对比 CPU 程序和 OpenCL 程序执行时间。

详细实现见代码。至于 CPU 程序，显然运行在 CPU 上的 OpenCL 程序也是 CPU 程序。

选取 Libbie 和 Microsoft Wallpaper 为输入图像。格式为 BMP。图片顺时针旋转 60 度；运行 10 次取平均值。

> 前者为 LibreOffice 吉祥物，后者为 Windows Insider 5 周年纪念壁纸。

|图片|尺寸|时间（OpenCL）|时间（CPU）|
|---|---|---|---|
|Libbie|$2391 \times 2816$|534.78 ms|984.08 ms|
|Microsoft Wallpaper|$4092 \times 2098$|563.21 ms|1023.21 ms|
