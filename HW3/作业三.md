# 异构计算 作业三

> 编写浮点矩阵乘法的 OpenCL 程序，采用 local memory 进行性能优化。基本要求：
>
> 1. 参考提供了样例代码，调试并正确运行；
> 2. 改变矩阵大小（1024×1024，8192×8192……），运行程序并记录和对比执行时间变化；
> 3. 将 `float` 数组改为 `double`，运行程序并记录和对比执行时间变化；
> 4. 采用分块策略，应用 local memory 存储对 kernel 程序进行优化，运行程序并记录和对比执行时间变化。

详细实现见代码。下面比较执行时间。约定如下：

1. 对于每一个矢量长度，执行 10 次取平均值；
2. 时间以**微秒**计；
3. 对于 GPU，仅计入纯粹计算的时间，不计入数据搬运的时间。

结果取对数如下：

- 横轴：$\log_2 (\text{矢量长度})$；纵轴：$\log_2 (\text{执行时间})$
- 蓝色：CPU；黄色：GPU；绿色：CPU 与 GPU 之差值。

可见无论是否分块，`double` 的执行时间都约为 `float` 的两倍；而分块可以带来极大的性能提升，同样使用 `float`，未分块的执行时间是分块的四倍。
